## Key Design Decisions

1. **Private storage model**: Uses `internal_slots[PrivateBrandStore]` on each instance's PropertyBag. Stores a `Value::Object` whose bag.properties hold private field key:value pairs. This avoids modifying PropertyBag or ObjectData structs.
2. **Brand**: Unique `Symbol` per class declaration, stored in `ClassConstructorData.private_brand`.
3. **Parent-child collision**: `P.#x` and `C.#x` are separate. Brand check ensures each class accesses only its own storage. Implementation: store multiple brand-name maps under the `PrivateBrandStore` internal slot (maybe as a Map keyed by brand symbol ID).
4. **Private methods**: Stored per-class in `ClassConstructorData.private_methods`. Brand check required before call. NOT yet populated in `create_class` (needs private method collection during member loop).
