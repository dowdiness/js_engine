# Host Internal Slots Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Give embedders an opaque `HostSlotKey` + get/set/delete/has API so host bookkeeping on JS objects is invisible to user JavaScript (#518).

**Architecture:** Add `PropertyBag.host_slots : Map[Int, Value]` beside engine `internal_slots`. Keys are opaque `HostSlotKey` values allocated by `HostSlotKey::reserve()` (monotonic `Ref[Int]`, abort on exhaustion). Accessors live in new `interpreter/runtime/host_slots.mbt`; engine `InternalSlotKey` is unchanged.

**Tech Stack:** MoonBit, `NEW_MOON_MOD=0 moon` CLI

## Global Constraints

- Spec: `docs/superpowers/specs/2026-07-11-host-internal-slots-design.md` (Approved)
- Always prefix moon commands: `NEW_MOON_MOD=0 moon ŌĆ”`
- After every file edit that changes types: `NEW_MOON_MOD=0 moon check` before the next file
- Do not re-export through root `@js_engine` facade in this slice
- Do not extend `InternalSlotKey` or share one map with engine slots
- `PropertyBag` stays `pub(all)` ŌĆö boundary is API convention, not field opacity
- `reserve()` must never wrap; abort on exhaustion
- No test262 changes expected
- Before commit: `NEW_MOON_MOD=0 moon info && NEW_MOON_MOD=0 moon fmt`

- [ ] **Step 2: Create `interpreter/runtime/host_slots.mbt`**

```moonbit
///| Embedder-owned host slots (#518). Separate from engine `internal_slots`.

///|
let host_slot_id_counter : Ref[Int] = { val: 0 }

///|
/// Embedder-owned slot identity. Construct only via `HostSlotKey::reserve`.
pub(all) struct HostSlotKey {
  priv id : Int
} derive(Eq, Hash, Debug)

///|
/// Allocate a unique key within this loaded runtime package instance.
/// Aborts if the allocator is exhausted ŌĆö never wraps.
pub fn HostSlotKey::reserve() -> HostSlotKey {
  let id = host_slot_id_counter.val
  // Last valid id is 0x7FFFFFFE; refuse to wrap into negatives / reuse.
  guard id < 0x7FFFFFFF else { abort("HostSlotKey::reserve: exhausted") }
  host_slot_id_counter.val = id + 1
  { id, }
}

///|
pub fn PropertyBag::get_host_slot(self : PropertyBag, key : HostSlotKey) -> Value? {
  self.host_slots.get(key.id)
}

///|
pub fn PropertyBag::set_host_slot(
  self : PropertyBag,
  key : HostSlotKey,
  value : Value,
) -> Unit {
  self.host_slots[key.id] = value
}

///|
pub fn PropertyBag::delete_host_slot(self : PropertyBag, key : HostSlotKey) -> Unit {
  self.host_slots.remove(key.id)
}

///|
pub fn PropertyBag::has_host_slot(self : PropertyBag, key : HostSlotKey) -> Bool {
  match self.host_slots.get(key.id) {
    Some(_) => true
    None => false
  }
}

///|
pub fn get_host_slot(data : ObjectData, key : HostSlotKey) -> Value? {
  data.bag.get_host_slot(key)
}

///|
pub fn set_host_slot(data : ObjectData, key : HostSlotKey, value : Value) -> Unit {
  data.bag.set_host_slot(key, value)
}

///|
pub fn delete_host_slot(data : ObjectData, key : HostSlotKey) -> Unit {
  data.bag.delete_host_slot(key)
}

///|
pub fn has_host_slot(data : ObjectData, key : HostSlotKey) -> Bool {
  data.bag.has_host_slot(key)
}
```

Note: `key.id` is readable here because `host_slots.mbt` is the same package as the `priv` field. External packages must use the accessors.

```moonbit
///|
test "host slot is invisible to Object.keys for-in and JSON.stringify" {
  let interp = new_interpreter()
  let prog = @parser.parse("var o = { visible: 1 };")
  let _ = interp.run(prog.stmts)
  let o = interp.global.get("o")
  let key = @runtime.HostSlotKey::reserve()
  match o {
    Object(data) => @runtime.set_host_slot(data, key, Number(99.0))
    _ => fail("expected Object for o")
  }
  let check = @parser.parse(
    #|var keys = Object.keys(o).join(",");
    #|var enumed = [];
    #|for (var k in o) enumed.push(k);
    #|var json = JSON.stringify(o);
    #|var own = o.host_secret;
    #|console.log(keys);
    #|console.log(enumed.join(","));
    #|console.log(json);
    #|console.log(typeof own);
  )
  let _ = interp.run(check.stmts)
  json_inspect(interp.host.output, content=[
    "visible",
    "visible",
    "{\"visible\":1}",
    "undefined",
  ])
}
```
